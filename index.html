1- Columns' names to be adjusted for easiness of analysis.
2- Item ID data type to be changed from number to text.

4- To add a new column to calculate the duration from Stage 1 till Stage 4
5- Explore what stages that work in parallel and what stages are depending on others.
6- If there was independices, more durations needs to be added to breakdown the performance of each team.

In [490]:
def duration(x, y, condition):
    task_duration = []
    for i in range(x.shape[0]):
        if condition[i]:
            task_duration.append(y[i] - x[i])
        else:
            task_duration[i].append("-")
    return task_duration
